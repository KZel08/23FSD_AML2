# Simple Counter SPA

This is a minimal Single Page Application demonstrating a counter using React's `useState` hook.

Files:
- index.html — Vite entry HTML
- src/main.jsx — app entry
- src/App.jsx — the React component implementing the counter

Run with Vite:

```bash
cd SPA_Counter
npm install
npm run dev
# open the URL printed by Vite (usually http://localhost:5173)
```

Build and preview:

```bash
npm run build
npm run preview
```
